# CAN IDS Pipeline

A toolkit for developing, evaluating, and evading Controller Area Network (CAN) Bus Network Intrusion Detection Systems (NIDS).

Features
- Add datasets or use existing datasets
- Plug-in IDS models (your own or built-in)
- Train and test IDS models
- Apply adversarial/real attacks and observe results
- Modular experiment workflow: dataset → IDS → attack → evaluation
